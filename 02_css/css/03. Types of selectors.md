## 1. Simple Selectors

These select elements based on their name, ID, class, or all elements.
- **Element (Type) selector**
    - Selects by HTML tag name, like `p`, `div`, `h1`, etc.
    - Example: `p { color: blue; }` → styles all `<p>` elements.
- **Class selector**
    - Selects elements with a given `class` attribute (use `.`).
    - Example: `.my-class { font-size: 16px; }` → styles all elements with `class="my-class"`.
- **ID selector**
    - Selects a single element with a given `id` attribute (use `#`).
    - Example: `#header { background: black; }` → styles the element with `id="header"`.
- **Universal selector**
    - Selects every element on the page (use `*`).
    - Example: `* { margin: 0; }` → removes margin from all elements

## 2. Group Selector

- Combines multiple selectors into one rule using commas.
- Example:
    `h1, h2, h3 {   font-family: Arial; }`
    → applies the style to all `<h1>`, `<h2>`, and `<h3>` elements.
