The `display` property controls element layout behavior, determining if elements stack vertically, flow horizontally, or form modern layouts like flex/grid.[](https://www.w3schools.com/cssref/pr_class_display.php)
## Core Values

| Value          | Behavior                      | Use Case                 |
| -------------- | ----------------------------- | ------------------------ |
| `block`        | Full width, new line          | `<div>`, `<p>`, headings |
| `inline`       | Text flow, no line breaks     | `<span>`, `<strong>`     |
| `inline-block` | Inline + width/height support | Buttons, images in text  |
| `none`         | Completely hidden             | Toggle visibility        |
| `flex`         | Flexbox container             | Responsive rows/columns  |
| `grid`         | CSS Grid container            | Complex 2D layouts       |

### Flex row
```css
.container{
	background-color: aqua;
	width: 100px;
	height: 100p;
	display: flex;
	flex-direction: row;
	/* row setted */
	justify-content: center;
	/* row axis center */
	align-items: center;
	/* row align center */
	gap: 10px;
	/* this will provide the gap between */
}
```
