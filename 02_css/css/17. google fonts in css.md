### **Step 1: Choose and Embed the Font**

First, you find a font you like on the Google Fonts website, select the styles/weights you want, and copy the provided embed code. You have two main ways to bring that code into your project:

**Method A: The HTML `<link>` Tag (Recommended)** This is generally faster because the browser starts downloading the font at the exact same time it reads your HTML. You place these tags inside your HTML's `<head>`.



```HTML
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet">
```

**Method B: The CSS `@import` Rule** If you only have access to a CSS file and can't edit the HTML, you can import the font directly into your stylesheet. This must go at the **very top** of your CSS file.



```CSS
@import url('https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap');
```

---

### **Step 2: Apply it with `font-family`**

Once the font is embedded, you apply it to your elements just like any other CSS property. You should always include a generic "fallback" category (like `sans-serif` or `serif`) just in case the Google Font fails to load.



```CSS
body {
  font-family: 'Roboto', sans-serif;
}
```

---

### **Putting It Together (Runnable Example)**

In this example, we will use an `@import` rule to bring in two very distinct Google Fonts: **'Oswald'** (a bold, condensed font perfect for headings) and **'Open Sans'** (a clean, readable font perfect for paragraphs).



```HTML
<!DOCTYPE html>
<html lang="en">
<head>
    <style>
        /* 1. Import the fonts at the very top of your CSS */
        @import url('https://fonts.googleapis.com/css2?family=Open+Sans:ital,wght@0,400;0,700;1,400&family=Oswald:wght@500;700&display=swap');

        /* 2. Set the default font for the whole page */
        body {
            font-family: 'Open Sans', sans-serif;
            background-color: #f4f4f9;
            color: #333;
            padding: 40px;
            max-width: 600px;
            margin: 0 auto;
            line-height: 1.6;
        }

        /* 3. Apply the heading font */
        h1, h2 {
            font-family: 'Oswald', sans-serif;
            text-transform: uppercase; /* Oswald looks great in all caps */
            letter-spacing: 1px;
            color: #2c3e50;
        }

        h1 {
            font-size: 3rem;
            margin-bottom: 10px;
        }

        .highlight {
            font-weight: 700;
            color: #e74c3c;
        }

        .italic-text {
            font-style: italic; /* Uses the imported italic style of Open Sans */
        }
    </style>
</head>
<body>

    <h1>This is Oswald</h1>
    
    <h2>A Great Font for Headers</h2>
    
    <p>This paragraph is using <strong>Open Sans</strong>. Notice how much easier it is to read in long blocks of text compared to the heading font. When selecting fonts, it's generally a good rule of thumb to pair a stylized heading font with a very neutral, highly legible body font.</p>
    
    <p class="italic-text">Because we imported the italic and bold weights of Open Sans, the browser renders <span class="highlight">these styles</span> perfectly without having to fake them.</p>

</body>
</html>
```
