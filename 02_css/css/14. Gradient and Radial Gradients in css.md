### **1. Linear Gradients (`linear-gradient`)**

A linear gradient transitions colors along a straight line. You can control the direction of this line using either keywords (like `to bottom`, `to right`, `to bottom right`) or specific angles (like `45deg`, `180deg`).

**The Basic Syntax:**

CSS

```
background-image: linear-gradient(direction, color1, color2, ...);
```

**How it works:**

- If you don't specify a direction, the default is top-to-bottom.
    
- You can add as many colors as you want (called "color stops").
    
- You can also specify exactly _where_ a color should stop (e.g., `blue 20%`).
    

**Example: A diagonal sunrise**



```CSS
.linear-box {
  /* Starts from the bottom-left and moves to the top-right */
  background-image: linear-gradient(to top right, #ff7e5f, #feb47b);
}
```

---

### **2. Radial Gradients (`radial-gradient`)**

A radial gradient transitions colors outward from a central starting point (origin), rather than along a straight line.

**The Basic Syntax:**



```CSS
background-image: radial-gradient(shape size at position, color1, color2, ...);
```

**How it works:**

- **Shape:** It defaults to an `ellipse` (stretching to fit the container), but you can force it to be a perfect `circle`.
    
- **Position:** It defaults to `center`, but you can move the origin point (e.g., `at top left` or `at 20% 50%`).
    
- **Size:** Controls how far the gradient extends (e.g., `closest-side`, `farthest-corner`).
    

**Example: A glowing orb effect**


```CSS
.radial-box {
  /* A perfect circle radiating from the exact center */
  background-image: radial-gradient(circle at center, #00d2ff, #3a7bd5);
}
```

### **Putting It Together (Runnable Example)**

Here is a complete HTML and CSS snippet you can test in any browser or code editor:



```HTML
<!DOCTYPE html>
<html lang="en">
<head>
    <style>
        .container {
            display: flex;
            gap: 20px;
            padding: 20px;
        }

        .box {
            width: 200px;
            height: 200px;
            border-radius: 12px;
            color: white;
            font-family: sans-serif;
            display: flex;
            align-items: center;
            justify-content: center;
            font-weight: bold;
            text-align: center;
        }

        /* Linear Gradient Example */
        .linear {
            background-image: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
        }

        /* Radial Gradient Example */
        .radial {
            background-image: radial-gradient(circle at 30% 30%, #ff9a9e 0%, #fecfef 99%, #fecfef 100%);
            color: #333; /* Dark text for a lighter background */
        }
    </style>
</head>
<body>

<div class="container">
    <div class="box linear">Linear<br>(135deg)</div>
    <div class="box radial">Radial<br>(Circle at 30% 30%)</div>
</div>

</body>
</html>
```
