CSS Margin Notes
CSS margin creates space outside an element’s border, separating it from surrounding elements. It supports lengths like px, percentages, `auto`, and negative values.
Individual Properties
Use these for specific sides:
	•	`margin-top`: Space above the element.
	•	`margin-right`: Space to the right.
	•	`margin-bottom`: Space below.
	•	`margin-left`: Space to the left.

```css
p {
  margin-top: 100px;
  margin-bottom: 100px;
  margin-right: 150px;
  margin-left: 80px;
}

```

Shorthand Syntax
The `margin` property combines all sides clockwise (top, right, bottom, left).

|Values|Meaning|Example|
|---|---|---|
|1 value|All sides|`margin: 25px;`|
|2 values|Top/bottom, left/right|`margin: 25px 50px;`|
|3 values|Top, left/right, bottom|`margin: 25px 50px 75px;`|
|4 values|Top, right, bottom, left|`margin: 25px 50px 75px 100px;`[](https://www.w3schools.com/css/css_margin.asp)|

## Margin Collapse

Adjacent vertical margins collapse to the largest value, ignoring smaller ones. This doesn't apply to horizontal margins, floated, or absolutely positioned elements. Negative margins reduce the space.
