
These properties control how the image occupies space and how it behaves when the container doesn't match the image's aspect ratio.

- **`width` / `height`**: Sets the dimensions. Pro tip: set one to `auto` to maintain the aspect ratio.
    
- **`max-width: 100%`**: A staple for responsive design; it prevents images from ever being wider than their parent container.
    
- **`object-fit`**: Decides how the image "stretches" or "crops" to fit its box.
    
    - `cover`: Fills the box, cropping the edges (most common).
        
    - `contain`: Shows the whole image, adding "letterboxing" if needed.
        
    - `fill`: Stretches the image (usually looks bad).
