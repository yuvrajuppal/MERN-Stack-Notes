There are three main types of CSS association (ways to link CSS to HTML):

## 1. Inline CSS

- You write CSS directly inside an HTML element using the `style` attribute.
- Example:
```html
    <p style="color: blue; font-size: 16px;">This is blue text.</p>
```    
- Best for quick, one‑time styling of a single element, but not good for large projects.
## 2. Internal (Embedded) CSS

- You write CSS rules inside a `<style>` block in the `<head>` section of an HTML page.
- Example    
```html
<head> 
	<style>   
	   p { color: red; font-size: 18px; }    
	   h1 { text-align: center; } 
	</style>
</head>
```    
- Useful for styling a single HTML page, but styles can’t be reused on other pages.


## 3. External CSS

- You write CSS in a separate `.css` file (e.g., `style.css`) and link it to HTML using the `<link>` tag.
- Example:
    ```html
      <head>   <link rel="stylesheet" href="style.css"> </head>
    ```
    
- **Recommended for most projects** because it keeps code clean, allows reuse across pages, and makes updates easier.

## Quick comparison

| Type     | Where it’s written                             | Best for                           |
| -------- | ---------------------------------------------- | ---------------------------------- |
| Inline   | Inside HTML tag, using `style=""`              | One-off styling of single elements |
| Internal | Inside `<style>` in `<head>`                   | Single HTML page                   |
| External | In a separate `.css` file, linked via `<link>` | Multiple pages / large projects    |

External CSS is what professionals and frameworks (like Bootstrap, Tailwind) mostly use.
