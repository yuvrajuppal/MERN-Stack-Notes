CSS borders create a visual outline around elements like divs or paragraphs, customizable via width, style, and color. The `border` shorthand property combines these for efficiency

### Border Syntax
The basic syntax is `border: width style color;`, where width uses units like `px`, style includes options like `solid` or `dashed`, and color can be named or hex.
	•	Common styles: `solid`, `dotted`, `dashed`, `double`, `groove`, `ridge`, `inset`, `outset`, `none`.
	•	Example: `border: 2px solid black;` applies a 2-pixel thick solid black border to all sides.

### Individual Sides
Target specific sides with `border-top`, `border-right`, `border-bottom`, or `border-left`.

```css
div {
  border-top: 1px solid red;
  border-bottom: 3px dashed blue;
}
```
This sets a thin red top border and thicker dashed blue bottom border.


### Full Example
Here’s a complete HTML/CSS demo for a styled box:

```html
<!DOCTYPE html>
<html>
<head>
	<style>
		.box {
		  border: 4px solid #333;
		  padding: 20px;
		  text-align: center;
		  width: 200px;
		}
	</style>
</head>
<body>
	<div class="box">Border Example</div>
</body>
</html>
```
