CSS `border-radius` rounds the corners of an element’s border, creating smooth, curved effects instead of sharp 90-degree angles. It works on any element with a border, like buttons, cards, or images, and accepts values in pixels, percentages, or other units.

Basic Syntax
Use `border-radius: value;` to apply uniform rounding to all four corners. Values can be `px`, `em`, `%`, or keywords like `initial`.
	•	Single value: `border-radius: 10px;` rounds all corners equally.
	•	Two values: 
```css
border-radius: 10px 20px;
```
 applies 10px to top-left/bottom-right, 20px to top-right/bottom-left.
	•	Three or four values follow clockwise order: top-left, top-right, bottom-right, bottom-left.

Percentage Values
Percentages are relative to the element’s width (horizontal radius) or height (vertical), enabling elliptical curves. 



```css
border-radius: 50%;
```
on a square element creates a perfect circle

### Full Example
```html
<!DOCTYPE html>
<html>
<head>
<style>
.card {
  border: 3px solid #333;
  padding: 20px;
  width: 200px;
  background: skyblue;
  border-radius: 15px; /* All corners rounded */
}

.circle {
  width: 100px;
  height: 100px;
  background: coral;
  border: 4px solid navy;
  border-radius: 50%; /* Perfect circle */
}
</style>
</head>
<body>
<div class="card">Rounded Card</div>
<div class="circle"></div>
</body>
</html>
```
