In CSS, `padding` is the space **inside** an element, between its content and its border (it creates “inner” space)

## What padding does
- Adds invisible space around the content (text, image, etc.) but **inside** the element’s border.
- Does not push other elements away; that’s the job of `margin` (which is outside the border).
- Background color / image of the element also extends into the padding area.


## How to set padding
You can set padding in four ways:
## 1. Shorthand `padding` (one line)
```css
div {
  padding: 20px;           /* all 4 sides */
  padding: 10px 20px;      /* top/bottom = 10px, left/right = 20px */
  padding: 10px 20px 30px; /* top=10, right/left=20, bottom=30 */
  padding: 10px 20px 30px 40px; /* top=10, right=20, bottom=30, left=40 */
}
```

## 2. Individual properties
```css
div {
  padding-top:    20px;
  padding-right:  15px;
  padding-bottom: 20px;
  padding-left:   10px;
}
```
