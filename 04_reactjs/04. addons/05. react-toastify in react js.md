### Quick Implementation

Since you are building that **Gradient Card**, adding a toast notification is a much smoother way to tell the user "CSS Copied!" than using a standard browser alert.

Here is how to set it up in your project:

**Wrap your App:** In your `App.js` or `main.jsx`, import the container and the CSS.
```jsx
import { ToastContainer } from 'react-toastify';
import 'react-toastify/dist/ReactToastify.css';

function App() {
  return (
    <div>
      <YourGradientComponent />
      <ToastContainer theme="dark" position="bottom-right" />
    </div>
  );
}
```

**Trigger the Toast in your Card:**
```js
import { toast } from 'react-toastify';

// Inside your GradientCard component:
const copyToClipboard = () => {
  window.navigator.clipboard.writeText(`background-image: ${gradientString};`);

    style: { background: '#24233a', color: '#fff' }
```
