
LocalStorage in React persists data in the browser across sessions without expiration. Use it with `useState` + `useEffect` for simple cases or a custom hook for reusability.
## Basic Implementation
Load from localStorage on mount, save on state change:


```jsx
import { useState, useEffect } from 'react';

function SimpleCounter() {
  const [count, setCount] = useState(0);

  // Load count when component mounts
  useEffect(() => {
    const saved = localStorage.getItem('count');
    if (saved) setCount(Number(saved));
  }, []);

  // Save count when it changes
  useEffect(() => {
    localStorage.setItem('count', count);
  }, [count]);

  return (
    <div>
      <h1>Count: {count}</h1>
      <button onClick={() => setCount(count + 1)}>+1</button>
    </div>
  );
}

```


## Complete Counter Example with Clear Button
To clear **all** localStorage data, use one simple line:
```jsx
import { useState, useEffect } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  useEffect(() => {
    const saved = localStorage.getItem('count');
    if (saved) setCount(Number(saved));
  }, []);

  useEffect(() => {
    localStorage.setItem('count', count);
  }, [count]);

  const clearAllStorage = () => {
    localStorage.clear();  // âœ… Clears EVERYTHING in localStorage
    setCount(0);           // Reset counter too
  };

  return (
    <div>
      <h1>Count: {count}</h1>
      <button onClick={() => setCount(count + 1)}>+1</button>
      <button onClick={clearAllStorage}>Clear Storage</button>
    </div>
  );
}
```
