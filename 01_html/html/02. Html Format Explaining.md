just create 
myfirst.html
in the vscode folder

```html
<DOCTYPE html>
<!-- this check that browser support html5 -->
	<html lang="en">
	<!-- this tell the the html is in english -->
	<head>
		<!-- this is the head of the html
		can be called the brain of the html -->
		<meta charset="UTF-8">
		<meta name="viewport" content="width=device-width, initial-scale=1.0">
		<!-- this is the meta tag used to make the site responsive -->
		<title>Document</title>
		<!-- this is the title of the html
		it is visible in the tab -->
	</head>
	<body>
	<!-- this is the body of the html
	can be called the body of the html
	all the ui part is done here -->
	</body>
</html>
```
