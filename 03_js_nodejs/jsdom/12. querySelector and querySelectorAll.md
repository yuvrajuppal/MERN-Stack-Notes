`querySelector` and `querySelectorAll` are modern JavaScript methods used to find HTML elements in a webpage using CSS selectors (like class, id, tag name, etc.).


They are part of the DOM API and are much more powerful and flexible than older methods like `getElementById()` or `getElementsByClassName()`.

## `querySelector(selector)`

- Returns the **first element** in the document (or inside a specific element) that matches the given CSS selector.    
- If no element matches, it returns `null`.
- Useful when you want to select just one element (like a form, a button, or a specific div).
```
element = document.querySelector(selector);
// or
element = someElement.querySelector(selector);
```

```js
// Get the first <p> with class "highlight"
const firstHighlight = document.querySelector('.highlight');
console.log(firstHighlight.textContent); // "First paragraph"

// Get an element by id
const container = document.querySelector('#container');
container.style.border = '2px solid red';

// Get the first <p> inside the container
const firstParagraph = container.querySelector('p');
firstParagraph.style.color = 'blue';

```


## `querySelectorAll(selector)`

- Returns a **NodeList** (a static, array‑like collection) of **all elements** that match the CSS selector.
- If no elements match, it returns an empty NodeList (not null).
- Useful when you want to work with multiple elements (like all buttons, or all list items).

```js
const elements = document.querySelectorAll(selector);
// or
const elements = someElement.querySelectorAll(selector);
```

```html
<ul id="fruits">
  <li class="fruit">Apple</li>
  <li class="berry">Blueberry</li>
  <li class="fruit">Orange</li>
  <li class="berry">Strawberry</li>
</ul>
```


## Selecting by different CSS selectors

Both methods accept any CSS selector, so you can combine them powerfully:
```js
// Get all elements with class "fruit"
const fruits = document.querySelectorAll('.fruit');
console.log(fruits.length); // 2

// Change style of all "fruit" items
fruits.forEach(fruit => {
  fruit.style.color = 'orange';
});

// Get all <li> elements inside #fruits
const allItems = document.querySelectorAll('#fruits li');
allItems.forEach(item => {
  item.style.backgroundColor = '#f0f0f0';
});
```

## Key differences

| Feature             | `querySelector`                    | `querySelectorAll`                                       |
| ------------------- | ---------------------------------- | -------------------------------------------------------- |
| Returns             | First matching element (or `null`) | NodeList of all matches (never `null`)                   |
| Return type         | Single Element                     | NodeList (array‑like)                                    |
| How to access items | Directly as an object              | Loop over NodeList (with `forEach`, `for`, spread, etc.) |
| Use case            | Modify one element (e.g. a form)   | Modify many elements (e.g. all buttons)                  |
