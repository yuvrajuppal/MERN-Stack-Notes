The `map()` method in JavaScript creates a new array by applying a callback function to each element of the original array, transforming values without mutating the source.
```js
const numbers = [1, 2, 3, 4];
const doubled = numbers.map((n)=>{
return  n * 2
});
console.log(doubled);  // [2, 4, 6, 8]
console.log(numbers);  // [1, 2, 3, 4] (unchanged) [web:52]

```

```js
const users = [{name: 'John'}, {name: 'Jane'}];
const greetings = users.map((user) =>{ 
return `Hello ${user.name}!`}
);
// ["Hello John!", "Hello Jane!"] [web:53]

```

const doubled = numbers.map(n => n * 2);
