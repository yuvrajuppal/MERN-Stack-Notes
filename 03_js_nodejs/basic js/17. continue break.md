
`break` exits a while loop completely when encountered, while `continue` skips the rest of the current iteration and jumps back to the condition check.

```js
while (condition) {
  if (someCondition) break;    // Exit loop entirely
  if (otherCondition) continue; // Skip to next iteration
  // rest of loop body
}
```

```js
let n = 0;
while (n <= 6) {
  if (n === 3) { n++; continue; }  // Skip logging 3
  console.log(n);                  // 0 1 2 4 5 6
  n++;
  if (n === 6) break;              // Stop at 6
}
```

`break` and `continue` work identically in `for` loops as in `while` loops—`break` exits completely, `continue` skips to the next iteration after the update step.[](https://www.geeksforgeeks.org/javascript/javascript-break-and-continue/)​

```js
for (let i = 0; i < 5; i++) {
  if (someCondition) break;     // Jump to after loop
  if (otherCondition) continue; // Skip body, do i++, check condition
  // rest of body
}

```

```js
for (let i = 0; i < 6; i++) {
  if (i === 2) continue;    // Skip logging 2
  if (i === 4) break;       // Stop after logging 0,1,3
  console.log(i);           // 0, 1, 3
}

```
