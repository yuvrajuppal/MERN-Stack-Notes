
The `while` loop in JavaScript executes a block of code repeatedly as long as a specified condition evaluates to `true`.[](https://www.w3schools.com/js/js_loop_while.asp)

```js
while (condition) {
  // code block to execute
}
```

```js
let i = 0;
while (i < 5) {
  console.log(i);  // Logs: 0, 1, 2, 3, 4
  i++;
}

```


The `do...while` loop in JavaScript executes its code block at least once before checking the condition, unlike a regular `while` loop.[](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Statements/do...while)


```js
do {
  // code block to execute
} while (condition);

```

```js
let i = 0;
do {
  console.log(i);  // 0, 1, 2, 3, 4
  i++;
} while (i < 5);
```
