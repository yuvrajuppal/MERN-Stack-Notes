## Mutator Methods (Modify Original Array)
- `push()` - adds to end
- `pop()` - removes from end
- `shift()` - removes from start
- `unshift()` - adds to start
```js
let removed = ll2.splice(0,2)
```
- `reverse()` - reverses order
- `fill()` - fills with value
```js
const arr = [1, 2, 3, 4, 5];
arr.fill(0, 2, 4); // fill from index 2 to 3 with 0
console.log(arr);   // → [1, 2, 0, 0, 5]
```



## Accessor Methods (Don't Modify Array)

- `length` - number of elements
- `concat()` - joins arrays
- `join()` - string separator
- `slice()` - extracts portion
- `toString()` - comma string
- `indexOf()` - first index
- `lastIndexOf()` - last index 
- `includes()` - contains value

# Example
```js
const fruits = ["Apple", "Orange", "Banana", "Orange", "Mango"];

// 1. length: Returns the number of elements
console.log(fruits.length); 
// Output: 5

// 2. concat(): Joins two or more arrays
const veggies = ["Carrot", "Potato"];
const food = fruits.concat(veggies);
console.log(food); 
// Output: ["Apple", "Orange", "Banana", "Orange", "Mango", "Carrot", "Potato"]

// 3. join(): Creates a string with a custom separator
console.log(fruits.join(" + ")); 
// Output: "Apple + Orange + Banana + Orange + Mango"

// 4. slice(): Extracts a portion (start index, end index - not inclusive)
const someFruits = fruits.slice(1, 3);
console.log(someFruits); 
// Output: ["Orange", "Banana"]

// 5. toString(): Converts array to a comma-separated string
console.log(fruits.toString()); 
// Output: "Apple,Orange,Banana,Orange,Mango"

// 6. indexOf(): Finds the FIRST index of a value
console.log(fruits.indexOf("Orange")); 
// Output: 1

// 7. lastIndexOf(): Finds the LAST index of a value
console.log(fruits.lastIndexOf("Orange")); 
// Output: 3

// 8. includes(): Checks if a value exists (returns true/false)
console.log(fruits.includes("Mango")); 
// Output: true
```
