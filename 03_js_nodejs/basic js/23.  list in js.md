## Mutator Methods (Modify Original Array)
- `push()` - adds to end
- `pop()` - removes from end
- `shift()` - removes from start
- `unshift()` - adds to start
```js
let removed = ll2.splice(0,2)
```
- `reverse()` - reverses order
- `sort()` - sorts elements
- `fill()` - fills with value
- `copyWithin()` - copies within array

## Accessor Methods (Don't Modify Array)

- `length` - number of elements
- `concat()` - joins arrays
- `join()` - string separator
- `slice()` - extracts portion
- `toString()` - comma string
- `toLocaleString()` - locale format
- `indexOf()` - first index
- `lastIndexOf()` - last index 
- `includes()` - contains value
## Iteration Methods (Loop/Process)

- `forEach()` - executes callback
- `map()` - transforms each
- `filter()` - conditions match
- `find()` - first match
- `findIndex()` - first index match
- `every()` - all pass test
- `some()` - any pass test
- `reduce()` - single value left
- `reduceRight()` - right to left
## Flattening Methods

- `flat()` - one level down
- `flatMap()` - map + flat
## New ES Methods

- `at()` - index (negative OK)
- `toSpliced()` - splice copy
- `toReversed()` - reverse copy
- `toSorted()` - sort copy
- `with()` - replace by index
