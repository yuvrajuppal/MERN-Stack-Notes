An IIFE (Immediately Invoked Function Expression) is a JavaScript function that runs as soon as it is defined, without needing a separate call like `myFunction()`.
```js
(function () {
  // code runs immediately
})();

```

```js
(() => {
  // code runs immediately
})();

```
