Strict equality in JavaScript uses the `===` operator to compare both the value and type of two operands without any type coercion.
```js
"hello" === "hello";  // true (same type, same value)
3 === 3;              // true
"3" === 3;            // false (string vs. number)
null === undefined;   // false
true === 1;           // false
NaN === NaN;          // false (NaN is never equal to itself)
{key: "value"} === {key: "value"};  // false (different objects)

```

The strict inequality operator `!==` in JavaScript checks if two operands are not equal in both value **and** type, without type coercion

```js
"hello" !== "hello";  // false (same type, same value)
3 !== 3;              // false
"3" !== 3;            // true (string vs. number)
null !== undefined;   // true
true !== 1;           // true
NaN !== NaN;          // true (NaN != itself)
{key: "value"} !== {key: "value"};  // true (different objects)
```
