The `filter()` method in JavaScript creates a new array with elements that pass a test implemented by a callback function, leaving the original array unchanged.â€‹

```
array.filter(callback(element, index, array))
```

```js
const numbers = [1, 2, 3, 4, 5, 6];
const evens = numbers.filter(num => num % 2 === 0);
console.log(evens);  // [2, 4, 6] [web:64]

```

```js
const users = [
  { name: 'John', age: 25 },
  { name: 'Jane', age: 17 },
  { name: 'Bob', age: 30 }
];
const adults = users.filter(user => user.age >= 18);
// [{ name: 'John', age: 25 }, { name: 'Bob', age: 30 }] 
```
