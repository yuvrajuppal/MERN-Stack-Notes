JavaScript logical operators include `&&` (AND), `||` (OR), and `!` (NOT), which combine or negate boolean values for conditional logic.[](https://www.w3schools.com/js/js_logical.asp)​
## Logical AND (&&)

The `&&` operator returns `true` only if both operands are truthy; otherwise, it short-circuits and returns the first falsy value (or the last value if all are truthy). For example, `true && false` evaluates to `false`, while `5 && 'hello'` returns `'hello'` due to short-circuit evaluation.​
```js
let age = 20;
let hasID = true;
if (age >= 18 && hasID) {
  console.log("Access granted");  // Outputs: Access granted
}
```
## Logical OR (||)

The `||` operator returns `true` if at least one operand is truthy, short-circuiting to the first truthy value (or the last if all falsy).​ Examples include `false || true` yielding `true`, and `0 || 'default'` returning `'default'`.[](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Expressions_and_operators)
​
```js
let user = null;
let name = user || "Guest";  // name becomes "Guest"
console.log(name);
```
## Logical NOT (!)

The `!` operator negates a single operand's boolean value, converting it to the opposite (`!true` is `false`, `!0` is `true`). Double negation (`!!value`) coerces to explicit boolean.​

```js
let isActive = true;
console.log(!isActive);  // false
```
