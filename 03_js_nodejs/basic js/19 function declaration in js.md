A JavaScript **function** is a reusable block of code that performs a specific task when called.

```js
function sayHello() {
  console.log("Hello World");
}

const greet = () => console.log("Hi!");


const sayWow = function() {
  console.log("wow");
} 


sayHello();  // Hello World
greet();     // Hi!
sayWow(); //wow

```
