The `forEach()` method in JavaScript executes a provided callback function once for each array element, iterating from index 0 to length-1

```js
array.forEach((element, index, array) => {
  // Code to run for each element
});

```

```js
const fruits = ['apple', 'banana', 'cherry'];
fruits.forEach(fruit => console.log(fruit));  
// apple, banana, cherry [web:41]

```

```js
fruits.forEach((fruit, index) => {
  console.log(`${index}: ${fruit}`);
});
// 0: apple, 1: banana, 2: cherry [web:46]

```
