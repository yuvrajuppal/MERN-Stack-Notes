
The `for` loop in JavaScript iterates over a block of code a specific number of times, using three expressions for initialization, condition, and update.[](https://www.w3schools.com/js/js_loop_for.asp)

```js
for (initialization; condition; update) {
  // code block to execute
}
```

```js
for (let i = 0; i < 5; i++) {
  console.log(i);  // 0, 1, 2, 3, 4
}
const fruits = ["apple", "banana"];
for (let i = 0; i < fruits.length; i++) {
  console.log(fruits[i]);
}
```
