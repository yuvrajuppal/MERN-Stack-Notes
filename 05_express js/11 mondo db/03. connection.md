In ES Modules, it’s cleaner to keep the connection logic separate. Create a file named `db.js` in your root or a `config` folder:

```JavaScript
import mongoose from "mongoose";

const connectDB = async () => {
  try {
    const conn = await mongoose.connect(process.env.MONGO_URI);
    console.log(`MongoDB Connected: ${conn.connection.host}`);
  } catch (error) {
    console.error(`Error: ${error.message}`);
    process.exit(1); // Exit process with failure
  }
};

export default connectDB;
```

---

### 2. Update your `.env`

Add your local MongoDB connection string to your `.env` file. Since you just installed it on your Mac, use the local address:


```Code snippet
SERVERPORT=5000
MONGO_URI=mongodb://127.0.0.1:27017/clientpulse 
```


### 3. Update your `server.js`

Now, import the connection function and call it before starting the server:

```JavaScript
import 'dotenv/config';
import app from "./app.js";
import connectDB from "./db.js"; // Import the connection

// Connect to Database
connectDB();

app.get('/', (req, res) => {
    res.send("server Started and Database Connected");
});

const ServerPort = process.env.SERVERPORT || 3000;

app.listen(ServerPort, () => {
    console.log(`server started at ${ServerPort}`);
});
```
