Building an API in Express typically follows the **RESTful** pattern, where we use `PUT` or `PATCH` for updates and `DELETE` for removals.

Below is a clean, modular example using the Controller-Route pattern in Express, followed by how to consume those endpoints in React using Axios.

---

## 1. The Express Backend

### Controller (`controllers/userController.js`)

The controller contains the logic. It interacts with your data (in this case, a dummy array) and sends the response.



```JavaScript
// Dummy data
let users = [
  { id: 1, name: "John Doe" },
  { id: 2, name: "Jane Smith" }
];

// UPDATE: Modify an existing user
export const updateUser = (req, res) => {
  const { id } = req.params;
  const { name } = req.body;

  const user = users.find((u) => u.id === parseInt(id));

  if (!user) {
    return res.status(404).json({ message: "User not found" });
  }

  user.name = name;
  res.status(200).json({ message: "User updated successfully", user });
};

// DELETE: Remove a user
export const deleteUser = (req, res) => {
  const { id } = req.params;
  const initialLength = users.length;
  
  users = users.filter((u) => u.id !== parseInt(id));

  if (users.length === initialLength) {
    return res.status(404).json({ message: "User not found" });
  }

  res.status(200).json({ message: "User deleted successfully" });
};
```

### Routes (`routes/userRoutes.js`)

The router maps the URL paths and HTTP methods to the controller functions.



```JavaScript
import express from 'express';
import { updateUser, deleteUser } from '../controllers/userController.js';

const router = express.Router();

// PUT is used for updates (replaces or modifies resource)
router.put('/users/:id', updateUser);

// DELETE is used for removal
router.delete('/users/:id', deleteUser);

export default router;
```

## 2. The React Frontend (Axios)

In React, we use **Axios** to send requests to these endpoints. Usually, you'll pass the `id` of the item you want to change or remove.

### UserComponent.jsx



```JavaScript
import React, { useState } from 'react';
import axios from 'axios';

const UserProfile = ({ userId }) => {
  const [newName, setNewName] = useState("");

  // Handler for Update
  const handleUpdate = async () => {
    try {
      const response = await axios.put(`http://localhost:5000/api/users/${userId}`, {
        name: newName
      });
      alert(response.data.message);
    } catch (error) {
      console.error("Update failed", error);
    }
  };

  // Handler for Delete
  const handleDelete = async () => {
    if (window.confirm("Are you sure you want to delete this user?")) {
      try {
        const response = await axios.delete(`http://localhost:5000/api/users/${userId}`);
        alert(response.data.message);
      } catch (error) {
        console.error("Delete failed", error);
      }
    }
  };

  return (
    <div>
      <input 
        type="text" 
        value={newName} 
        onChange={(e) => setNewName(e.target.value)} 
        placeholder="Enter new name"
      />
      <button onClick={handleUpdate}>Update Name</button>
      <button onClick={handleDelete} style={{ color: 'red' }}>Delete User</button>
    </div>
  );
};

export default UserProfile;
```

## Key Differences to Remember

|Feature|Update (`PUT`/`PATCH`)|Delete (`DELETE`)|
|---|---|---|
|**URL Params**|Requires `:id` to know _which_ record.|Requires `:id` to know _which_ record.|
|**Request Body**|Contains the new data (`req.body`).|Usually empty; only needs the ID.|
|**Status Code**|Usually `200 OK`.|Usually `200 OK` or `204 No Content`.|
