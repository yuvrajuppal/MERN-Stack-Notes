To round out your API, we’ll look at **GET** (retrieving data) and **POST** (creating data). In a professional Express setup, these follow the same Controller-Route pattern to keep your code clean and scalable.

### How the Data Flows

When a user interacts with your React app, Axios sends a request. The Express **Route** catches it and hands it off to the **Controller**, which processes the logic and sends back JSON.

---

## 1. The Express Backend

### Controller (`controllers/userController.js`)

The `getUsers` function returns the list, and `createUser` adds a new object to it.

```JavaScript
// Local data store (mimicking a database)
let users = [
  { id: 1, name: "Gemini" }
];

// GET: Fetch all users
export const getUsers = (req, res) => {
  res.status(200).json(users);
};

// POST: Create a new user
export const createUser = (req, res) => {
  const { name } = req.body;

  if (!name) {
    return res.status(400).json({ message: "Name is required" });
  }

  const newUser = {
    id: users.length + 1,
    name: name
  };

  users.push(newUser);
  res.status(201).json(newUser); // 201 means "Created"
};
```

### Routes (`routes/userRoutes.js`)

Notice how we use different HTTP methods (`get` and `post`) on the same base path.

```JavaScript
import express from 'express';
import { getUsers, createUser } from '../controllers/userController.js';

const router = express.Router();

router.get('/users', getUsers);
router.post('/users', createUser);

export default router;
```

## 2. The React Frontend (Axios)

In React, you typically use `useEffect` to **GET** data when the component loads and a function to **POST** data when a form is submitted.

### UserDashboard.jsx

```JavaScript
import React, { useState, useEffect } from 'react';
import axios from 'axios';

const UserDashboard = () => {
  const [users, setUsers] = useState([]);
  const [name, setName] = useState("");

  // 1. GET - Fetch users on component mount
  useEffect(() => {
    const fetchUsers = async () => {
      try {
        const res = await axios.get('http://localhost:5000/api/users');
        setUsers(res.data);
      } catch (err) {
        console.error("Error fetching users", err);
      }
    };
    fetchUsers();
  }, []);

  // 2. POST - Add a new user
  const handleAddUser = async (e) => {
    e.preventDefault();
    try {
      const res = await axios.post('http://localhost:5000/api/users', { name });
      // Update the UI by adding the new user to the existing list
      setUsers([...users, res.data]);
      setName(""); // Clear input
    } catch (err) {
      console.error("Error creating user", err);
    }
  };

  return (
    <div>
      <h2>User List</h2>
      <ul>
        {users.map(u => <li key={u.id}>{u.name}</li>)}
      </ul>

      <form onSubmit={handleAddUser}>
        <input 
          value={name} 
          onChange={(e) => setName(e.target.value)} 
          placeholder="New User Name" 
        />
        <button type="submit">Add User</button>
      </form>
    </div>
  );
};

export default UserDashboard;
```

### Pro-Tips for your API:

- **Body Parser:** Ensure your Express entry file (`app.js` or `server.js`) has `app.use(express.json());` so it can read the `req.body` from Axios.
    
- **Error Handling:** Always wrap your Axios calls in `try...catch` blocks to handle network errors or server crashes gracefully.
