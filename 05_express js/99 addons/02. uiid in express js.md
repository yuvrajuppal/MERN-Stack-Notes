**Using UUID v4 (most common):**
```js
const { v4: uuidv4 } = require('uuid');

app.post('/api/data', (req, res) => {
    const uuid = uuidv4();
    console.log(uuid); // e.g., '123e4567-e89b-12d3-a456-426614174000'
    
    res.json({
        id: uuid,
        message: 'UUID generated successfully'
    });
});


```
**ES6 import syntax:**
```js
import { v4 as uuidv4 } from 'uuid';
const uuid = uuidv4();
```
